# Weconds
Android Float Ball or Window for Time and Seconds 安卓全局悬浮窗动态时间显示。

Android Float Ball or Window for Time and Seconds.
Weconds是一个在操作其他App时提供时间提示的工具App，它提供了一个精确到秒的全局时间悬浮窗。
